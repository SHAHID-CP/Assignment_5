Question-2: What is the difference between innerHTML, innerText, and textContent ?

2.innerHTML,innerText এবং textContent এর মধ্যে পার্থক্য :

innerHTML :

এমন এক property যা দিয়ে documents এর কোন elements এর ভিতরের সব কনটেন্ট নিয়ে আসা যায় এবং নিয়ে আসা কনটেন্ট পরিবর্তন করা যায়।

innerText :

যা এমন এক property যা দিয়ে documents এর কোন elements এর ভিতরের text কে access করা যায় এবং পরিবর্তন করা যায়।

TextContent :

যা এমন এক property যা দিয়ে documents এর কোন elements এর ভিতরের সব text কে access করা যায় এবং পরিবর্তন করা যায় যদি সেটি css দিয়ে visibility:hidden বা display:none ও করা থাকে

Question-3: What is event delegation in the DOM?

3.DOM এ event delegation হলো :

Event delegation এমন এক প্রক্রিয়া যা কোনো parent এ event listener যোগ করলে ঐ parent এর সকল child এ ঐ event কাজ করবে।

Question-4: What is event bubbling in the DOM?

4.DOM এ event bubbling হলো :

Event bubbling এমন এক প্রক্রিয়া যেখানে একটি event তার target elements কে খুজবে এবং target element এ গিয়ে event ঘটবে। তারপর parent element গুলোর দিকে propagate করবে।

Question-5: How do you create, add, and remove elements using JavaScript?

5.Javascript দিয়ে যেভাবে elements create,add এবং remove করা যায় :

ID দিয়ে parent element ধরা :

const child=document.getElementById('parent');

New element create করা :

const newsec=document.createElement('section');

Parent element এ new element যোগ করা :

child.appendChild(newsec);

Main parent থেকে parent element কে remove করা :

child.parentNode.removeChild(child);
